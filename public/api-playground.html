API Playground

Trang nay dong vai tro nhu Postman online cho project. Ban co the chon san endpoint, sua body JSON, luu JWT token, gui request va xem response ngay tren trinh duyet.

Cau hinh

Tai khoan seed:
admin@example.com / 123456
staff@example.com / 123456
customer@example.com / 123456

Endpoint mau

Huong dan nhanh

1. Login 2. Save token 3. Chay CRUD 4. Tao order 5. Payment / status
  • Mo trang nay qua URL: /tools/api-playground.html khi server dang chay.
  • Chon endpoint trong cot trai, body mau se tu dien vao form.
  • Neu endpoint can dang nhap, trang se tu dong gan header Authorization: Bearer <token>.
  • Upload file khong thuc hien tren trang nay. Cac API upload nen demo bang Postman desktop voi form-data.
  • Danh sach body mau da duoc ghi chu o tung endpoint de ban chup man hinh va demo nhanh.
